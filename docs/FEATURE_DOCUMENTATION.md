## 🔗 Integration Guide

### Converter API
```javascript
// Client-side converter usage
import { convertPNGToSVG } from '@/lib/converters';

const result = await convertPNGToSVG(file, {
  threshold: 128,
  color: 'auto',
  smoothing: true
});
```

### Gallery Integration
```javascript
// Fetch gallery items
const galleryItems = await fetch('/api/gallery/heart-svg');
const designs = await galleryItems.json();
```
