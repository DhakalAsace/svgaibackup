## Optimizing Christmas SVGs for Different Uses

### File Size Optimization Techniques

1. **Simplify Complex Paths**
   - Reduce anchor points in curved designs
   - Merge similar shapes
   - Remove hidden elements

2. **Optimize Code Structure**
   ```xml
   <!-- Before optimization -->
   <circle cx="50" cy="50" r="10" fill="#FF0000"/>
   <circle cx="70" cy="50" r="10" fill="#FF0000"/>
   
   <!-- After optimization -->
   <g fill="#FF0000">
     <circle cx="50" cy="50" r="10"/>
     <circle cx="70" cy="50" r="10"/>
   </g>
   ```

3. **Use SVG Sprites**
   Combine multiple Christmas icons into one file for better performance

## Advanced Christmas SVG Techniques

### Creating Layered Effects
```xml
<svg viewBox="0 0 200 200">
  <defs>
    <filter id="shadow">
      <feDropShadow dx="2" dy="2" stdDeviation="2"/>
    </filter>
  </defs>
  <g filter="url(#shadow)">
    <!-- Christmas tree layers -->
    <polygon points="100,20 60,80 140,80" fill="#0F7938"/>
    <polygon points="100,60 50,120 150,120" fill="#165B33"/>
    <polygon points="100,100 40,160 160,160" fill="#1F3A28"/>
  </g>
</svg>
```
