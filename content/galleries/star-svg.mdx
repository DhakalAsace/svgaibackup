## Star Mathematics and Construction

### Creating Perfect Stars Programmatically

```javascript
function createStar(points, outerRadius, innerRadius) {
  let path = '';
  const angle = Math.PI / points;
  
  for (let i = 0; i < 2 * points; i++) {
    const radius = i % 2 === 0 ? outerRadius : innerRadius;
    const x = Math.cos(i * angle - Math.PI / 2) * radius;
    const y = Math.sin(i * angle - Math.PI / 2) * radius;
    path += `${i === 0 ? 'M' : 'L'} ${x},${y}`;
  }
  
  return path + 'Z';
}
```

## Frequently Asked Questions

### What's the best star SVG for rating systems?
Simple, filled five-pointed stars work best for ratings. Use a consistent design with clear filled/empty states. Our rating star template provides accessibility features and smooth animations.

### How do I create half-star ratings?
Use clip paths or gradients to show partial fills. Define the star shape once and apply different fill percentages. This maintains consistency while showing precise ratings like 3.5 stars.

### Can star SVGs be animated?
Yes! Stars are perfect for animations: twinkling effects, rotating bursts, growing/shrinking, color transitions, and drawing animations. CSS and JavaScript both work well for star animations.

### What's the ideal number of points for a star?
Five points are most recognizable and versatile. Six points work well for snowflakes or Jewish stars. Eight or more points create starburst effects. Choose based on your specific design needs.

### How do I make stars accessible?
Include proper ARIA labels, provide text alternatives for ratings, ensure sufficient color contrast, and make interactive stars keyboard accessible. Screen readers should understand the star's purpose and state.
