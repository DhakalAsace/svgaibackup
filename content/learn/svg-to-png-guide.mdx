## Why Convert SVG to PNG?

### Common Use Cases

SVG to PNG conversion serves numerous purposes in modern digital workflows:

**1. Icon Libraries and Design Systems**
When managing icon sets, you often need both SVG (for scalability) and PNG (for compatibility) versions. Proper conversion ensures every icon is available in both formats with consistent quality.

**2. Cross-Platform Development**
Different platforms have varying support for SVG. Mobile apps, email clients, and older browsers often require PNG alternatives. Strategic conversion creates these fallbacks efficiently.

**3. Performance Optimization**
While SVGs are great for scalability, PNGs can be more performant for complex graphics. Converting allows you to optimize performance for different use cases.

**4. Documentation and Presentation**
Technical documentation, presentations, and PDFs often work better with raster images. Conversion prepares your vector assets for these contexts.

**5. Social Media Assets**
Most social platforms prefer or require raster formats. Conversion helps prepare graphics for various platform requirements with the right dimensions and quality.

#### Batch Conversion Script

Create `batch-convert.js`:
```javascript
const fs = require('fs').promises;
const path = require('path');
const { glob } = require('glob');
const sharp = require('sharp');
const { optimize } = require('svgo');

async function batchConvertSVGtoPNG(inputPattern, outputDir, options = {}) {
  // Default options
  const defaults = {
    width: null,
    height: null,
    density: 300,
    background: { r: 0, g: 0, b: 0, alpha: 0 }
  };
  
  const settings = { ...defaults, ...options };
  
  // Ensure output directory exists
  await fs.mkdir(outputDir, { recursive: true });
  
  // Find all SVG files
  const files = await glob(inputPattern);
  
  console.log(`Found ${files.length} SVG files to convert`);
  
  // Process each file
  const results = await Promise.all(files.map(async (file) => {
    try {
      // Read SVG file
      const svgBuffer = await fs.readFile(file);
      
      // Optimize SVG (optional)
      const result = optimize(svgBuffer, {
        path: file,
        multipass: true
      });
      
      // Generate output filename
      const basename = path.basename(file, '.svg');
      const outputPath = path.join(outputDir, `${basename}.png`);
      
      // Convert to PNG
      await sharp(Buffer.from(result.data), { density: settings.density })
        .resize(settings.width, settings.height, { fit: 'inside' })
        .flatten({ background: settings.background })
        .png()
        .toFile(outputPath);
      
      console.log(`✓ Converted: ${file} → ${outputPath}`);
      return { success: true, input: file, output: outputPath };
      
    } catch (error) {
      console.error(`✗ Failed: ${file} - ${error.message}`);
      return { success: false, input: file, error: error.message };
    }
  }));
  
  // Summary
  const successful = results.filter(r => r.success).length;
  console.log(`\nConversion complete: ${successful}/${files.length} files converted successfully`);
  
  return results;
}

// Example usage
(async () => {
  // Convert all SVGs in current directory
  await batchConvertSVGtoPNG('*.svg', './png-output');
  
  // Convert with specific size
  await batchConvertSVGtoPNG('icons/*.svg', './png-icons', {
    width: 256,
    height: 256
  });
  
  // Convert with white background
  await batchConvertSVGtoPNG('logos/*.svg', './png-logos', {
    background: { r: 255, g: 255, b: 255, alpha: 1 }
  });
})();
```

### Using Cairosvg

```python
import os
import cairosvg
from pathlib import Path
import concurrent.futures
from tqdm import tqdm

def convert_svg_to_png(svg_path, output_dir, width=None, height=None, dpi=96):
    """Convert a single SVG file to PNG."""
    try:
        svg_file = Path(svg_path)
        png_file = output_dir / f"{svg_file.stem}.png"
        
        cairosvg.svg2png(
            file_obj=open(svg_path, 'rb'),
            write_to=str(png_file),
            output_width=width,
            output_height=height,
            dpi=dpi
        )
        return True, svg_path, png_file
    except Exception as e:
        return False, svg_path, str(e)

def batch_convert_svg_to_png(input_dir, output_dir, width=None, 
                           height=None, dpi=96, max_workers=4):
    """Batch convert SVG files to PNG with multiprocessing."""
    
    input_path = Path(input_dir)
    output_path = Path(output_dir)
    output_path.mkdir(parents=True, exist_ok=True)
    
    # Find all SVG files
    svg_files = list(input_path.glob("**/*.svg"))
    
    if not svg_files:
        print("No SVG files found!")
        return
    
    print(f"Found {len(svg_files)} SVG files to convert")
    
    # Convert with progress bar
    results = []
    with concurrent.futures.ThreadPoolExecutor(max_workers=max_workers) as executor:
        futures = {
            executor.submit(
                convert_svg_to_png, 
                svg_file, 
                output_path, 
                width, 
                height, 
                dpi
            ): svg_file 
            for svg_file in svg_files
        }
        
        for future in tqdm(concurrent.futures.as_completed(futures), 
                          total=len(svg_files), 
                          desc="Converting"):
            results.append(future.result())
    
    # Summary
    successful = sum(1 for r in results if r[0])
    print(f"\nConversion complete: {successful}/{len(svg_files)} successful")
    
    # Report errors
    errors = [r for r in results if not r[0]]
    if errors:
        print("\nErrors:")
        for _, file, error in errors:
            print(f"  {file}: {error}")

# Example usage
if __name__ == "__main__":
    batch_convert_svg_to_png(
        input_dir="./svg-icons",
        output_dir="./png-icons",
        width=256,
        height=256,
        dpi=144,
        max_workers=8
    )
```

### Advanced Python Script with Options

```python
import argparse
import os
from pathlib import Path
import cairosvg
import yaml
from PIL import Image
import logging

class SVGBatchConverter:
    def __init__(self, config_file=None):
        self.config = self.load_config(config_file)
        self.setup_logging()
    
    def load_config(self, config_file):
        """Load configuration from YAML file."""
        default_config = {
            'output_formats': {
                'small': {'width': 64, 'height': 64},
                'medium': {'width': 256, 'height': 256},
                'large': {'width': 512, 'height': 512}
            },
            'quality': 95,
            'dpi': 96,
            'background': 'transparent',
            'optimize': True
        }
        
        if config_file and Path(config_file).exists():
            with open(config_file, 'r') as f:
                user_config = yaml.safe_load(f)
                default_config.update(user_config)
        
        return default_config
    
    def setup_logging(self):
        """Setup logging configuration."""
        logging.basicConfig(
            level=logging.INFO,
            format='%(asctime)s - %(levelname)s - %(message)s',
            handlers=[
                logging.FileHandler('batch_conversion.log'),
                logging.StreamHandler()
            ]
        )
        self.logger = logging.getLogger(__name__)
    
    def convert_single(self, svg_path, output_path, size_name, dimensions):
        """Convert a single SVG file."""
        try:
            # Generate output filename
            svg_file = Path(svg_path)
            png_filename = f"{svg_file.stem}_{size_name}.png"
            png_path = output_path / png_filename
            
            # Convert SVG to PNG
            cairosvg.svg2png(
                file_obj=open(svg_path, 'rb'),
                write_to=str(png_path),
                output_width=dimensions['width'],
                output_height=dimensions['height'],
                dpi=self.config['dpi']
            )
            
            # Optimize if enabled
            if self.config['optimize']:
                self.optimize_png(png_path)
            
            self.logger.info(f"Converted: {svg_file.name} → {png_filename}")
            return True
            
        except Exception as e:
            self.logger.error(f"Failed to convert {svg_path}: {str(e)}")
            return False
    
    def optimize_png(self, png_path):
        """Optimize PNG file size."""
        try:
            img = Image.open(png_path)
            img.save(png_path, 'PNG', optimize=True, quality=self.config['quality'])
        except Exception as e:
            self.logger.warning(f"Failed to optimize {png_path}: {str(e)}")
    
    def batch_convert(self, input_dir, output_dir):
        """Batch convert all SVG files."""
        input_path = Path(input_dir)
        output_path = Path(output_dir)
        
        # Create output directory structure
        for size_name in self.config['output_formats']:
            (output_path / size_name).mkdir(parents=True, exist_ok=True)
        
        # Find all SVG files
        svg_files = list(input_path.rglob("*.svg"))
        total_conversions = len(svg_files) * len(self.config['output_formats'])
        
        self.logger.info(f"Starting batch conversion: {len(svg_files)} files, "
                        f"{len(self.config['output_formats'])} sizes each")
        
        successful = 0
        for svg_file in svg_files:
            for size_name, dimensions in self.config['output_formats'].items():
                if self.convert_single(
                    svg_file, 
                    output_path / size_name, 
                    size_name, 
                    dimensions
                ):
                    successful += 1
        
        self.logger.info(f"Batch conversion complete: "
                        f"{successful}/{total_conversions} successful")

# CLI usage
if __name__ == "__main__":
    parser = argparse.ArgumentParser(description='Batch convert SVG to PNG')
    parser.add_argument('input', help='Input directory containing SVG files')
    parser.add_argument('output', help='Output directory for PNG files')
    parser.add_argument('--config', help='Configuration file (YAML)')
    
    args = parser.parse_args()
    
    converter = SVGBatchConverter(args.config)
    converter.batch_convert(args.input, args.output)
```

### Gulp Task for Automated Conversion

```javascript
const gulp = require('gulp');
const svg2png = require('gulp-svg2png');
const rename = require('gulp-rename');
const imagemin = require('gulp-imagemin');

// Basic batch conversion task
gulp.task('svg-to-png', () => {
  return gulp.src('src/icons/*.svg')
    .pipe(svg2png())
    .pipe(imagemin([
      imagemin.optipng({ optimizationLevel: 5 })
    ]))
    .pipe(gulp.dest('dist/icons/png'));
});

// Multiple sizes batch conversion
gulp.task('svg-to-png-sizes', () => {
  const sizes = [16, 32, 64, 128, 256];
  
  return sizes.map(size => {
    return gulp.src('src/icons/*.svg')
      .pipe(svg2png({ width: size, height: size }))
      .pipe(rename(path => {
        path.basename += `-${size}px`;
      }))
      .pipe(gulp.dest('dist/icons/png'));
  });
});

// Watch for changes
gulp.task('watch', () => {
  gulp.watch('src/icons/*.svg', gulp.series('svg-to-png'));
});
```

### 4. Error Handling

Always implement proper error handling in batch scripts:

```bash
#!/bin/bash
# Batch convert with error logging

LOG_FILE="conversion_errors.log"
SUCCESS_COUNT=0
ERROR_COUNT=0

for file in *.svg; do
  if convert "$file" "${file%.svg}.png" 2>> "$LOG_FILE"; then
    ((SUCCESS_COUNT++))
  else
    ((ERROR_COUNT++))
    echo "Failed: $file" >> "$LOG_FILE"
  fi
done

echo "Conversion complete: $SUCCESS_COUNT successful, $ERROR_COUNT errors"
```

## Troubleshooting Common Issues

### Issue 1: SVG Text Not Rendering

**Solution**: Ensure fonts are embedded or converted to paths:
```bash
# Convert text to paths before batch conversion
inkscape input.svg --export-text-to-path --export-plain-svg=output.svg
```

### Issue 2: Transparency Lost

**Solution**: Explicitly set transparent background:
```bash
convert input.svg -background none -density 300 output.png
```

### Issue 3: Poor Quality Output

**Solution**: Increase density before resizing:
```bash
convert -density 300 input.svg -resize 256x256 output.png
```

### Issue 4: Memory Errors with Large Batches

**Solution**: Process files individually:
```bash
for file in *.svg; do
  convert "$file" "${file%.svg}.png"
  # Clear memory between conversions
  sync && echo 3 > /proc/sys/vm/drop_caches
done
```
